# LOTUS_DESIGNS_CAR_RADIO

## Features

- Carplay style UI
- Saves data via state bags to the vehicle
- Synced to all players in the car
- Music queue system
- Highly optimized (0.00-0.02)

## Dependency

- Onesync (uses state bags)
- [xsound](https://github.com/Xogy/xsound)
